# Explore

The **Explore** tab is your hub for browsing all server collections. Each section shows a count of items pulled from your PhotoPrism server.

:::info
Add a screenshot of the Explore screen as `docs/img/explore.png`.
:::

| Section | Description |
|---------|-------------|
| [Albums](./albums) | Your curated photo collections |
| [Folders](./folders) | Browse by directory structure |
| [Moments](./moments) | Auto-grouped by time & place |
| [Calendar](./calendar) | Organized by month and year |
