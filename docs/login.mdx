# Access

## Connect to Your PhotoPrism Server


<div style={{ display: 'flex', gap: '20px', justifyContent: 'center' }}>
  <img src={ServersImage} alt="Login Screen" style={{width: '300px', height: 'auto'}} />
  <img src={AddServerImage} alt="Add Server Screen" style={{width: '300px', height: 'auto'}} />
</div>


On the login screen, enter:

1. **Server URL**: Your PhotoPrism server address (e.g., `https://photos.example.com`)
2. **Username**: Your PhotoPrism username
3. **Password**: Your PhotoPrism password

Tap **"Connect and Save"** to authenticate.

:::warning

Use **https** for your PhotoPrism server URL to ensure secure connection.

:::
