### Info server

In this section you can see the information of the current server you are connected to.


### Appearance

In this section you can change the appearance of the app.


### Automatic Background Sync

1. Go to **Settings** tab
2. Enable **"Auto Sync Background"**
3. Set the interval for automatic sync.
4. PhotoSpectra will automatically sync new photos based on the set interval

### Managing Multiple Servers

1. Go to **Settings** tab
2. Tap **"Add Server"**
3. Enter new server credentials
4. Switch between servers as needed.
