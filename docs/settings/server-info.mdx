# Server Info

Tap **Server** in the Info section to see full details about your active connection.

## Connection

| Field | Description |
|-------|-------------|
| **Server URL** | The address of your PhotoPrism server |
| **Status** | `online` or `offline` |
| **Username** | Your PhotoPrism username |
| **Synced count** | Number of photos synced to this server |

## User

Details pulled from your PhotoPrism account:

- Display name, Email, Role
- **WebDAV** — shows if WebDAV is enabled on your account (required for uploads)
- Base path and Upload path on the server

## Session

Internal session data (Session ID, token type, scope). Useful for debugging authentication issues.

## Server Config

PhotoPrism server details: version, edition, mode, read-only status, membership, and server resource info (CPU cores, memory).
